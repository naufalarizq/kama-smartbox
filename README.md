# KAMA SMART FOOD BOX
## KAMA : Smart Food Box yang Terintegrasi dengan Teknologi Artificial Intelligence (AI) dan Internet of Things (IoT) untuk Mengurangi Limbah Makanan Secara Berkelanjutan Menuju Zero Waste

KAMA : Smart Food Box adalah kotak penyimpanan makanan cerdas yang dilengkapi dengan sistem analisis dan preservasi makanan berbasis IoT dan AI. Perangkat ini memanfaatkan berbagai sensor, seperti sensor suhu, kelembapan, serta sensor gas (COâ‚‚, metana, etilen, dan sejenisnya) untuk mendeteksi kondisi makanan secara real-time. Data dari sensor dianalisis menggunakan model klasifikasi berbasis AI untuk menentukan tingkat kesegaran makanan, memprediksi umur simpan, dan memberikan rekomendasi kepada pengguna terkait konsumsi maupun pengolahan makanan.
